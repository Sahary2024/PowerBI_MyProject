# HR Workforce Dashboard

## Project Overview
This Power BI dashboard provides a comprehensive overview of workforce data for HR analysis. It helps managers monitor key employee metrics, understand trends, and make informed decisions. The dashboard includes insights related to employee satisfaction, performance, demographics, and compensation.

## Key Features

- **Job Satisfaction:** Visualizes employee job satisfaction levels across departments, teams, or locations to identify engagement trends.  
- **Performance Rating:** Displays employee performance ratings to track top performers and areas for improvement.  
- **Employee Count by Gender:** Shows the distribution of employees by gender, supporting diversity analysis.  
- **Employee Count by Location:** Maps employee numbers based on latitude and longitude, providing a geographical view of workforce distribution.  
- **Average Monthly Income:** Calculates and visualizes the average monthly income across departments or employee categories.

## Purpose
This dashboard is designed to help HR teams:  
- Identify trends in employee satisfaction and performance.  
- Monitor workforce distribution and diversity.  
- Analyze compensation data for fairness and planning.
